Introduction to Pandas

Pandas is an open-source Python library used for data manipulation, analysis and preprocessing. It provides powerful data structures such as Series and DataFrame, making it easy to organize, clean and analyze structured data. Pandas is widely used in Machine Learning and Generative AI because models require well-organized datasets before training.

Features of Pandas
• Fast and efficient data handling.
• Supports CSV, Excel and JSON files.
• Handles missing values.
• Provides filtering, sorting and grouping.

Series
A Series is a one-dimensional labeled array capable of storing integers, strings and floating-point values.

Example:
import pandas as pd
s = pd.Series([10,20,30])

DataFrame
A DataFrame is a two-dimensional table consisting of rows and columns.

Example:
data = {"Name":["Aman","Riya"],"Marks":[90,85]}
df = pd.DataFrame(data)

Reading Data
df = pd.read_csv("students.csv")

Useful Functions
head() – Displays first five rows.
tail() – Displays last five rows.
info() – Shows dataset information.
describe() – Displays statistical summary.
