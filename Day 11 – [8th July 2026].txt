TOPICS COVERED (SUPERVISED AND UNSUPERVISED LEARNING)
Introduction

Machine Learning algorithms are broadly divided into supervised and unsupervised learning depending upon the availability of labelled data.

Supervised Learning

Uses labelled datasets where the correct output is already known.

Examples
• Linear Regression
• Logistic Regression
• Decision Trees

Applications
• House Price Prediction
• Email Spam Detection

Unsupervised Learning

Uses unlabelled datasets to identify hidden patterns.

Examples
• K-Means Clustering
• Hierarchical Clustering

Applications
• Customer Segmentation
• Market Basket Analysis

Comparison

Supervised → Labelled Data
Unsupervised → Unlabelled Data
