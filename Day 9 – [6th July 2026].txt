Introduction to Data Visualization

Data visualization is the graphical representation of data using charts and graphs. It helps identify patterns, trends and relationships in datasets before applying Machine Learning models.

Importance
• Better understanding of data
• Detect trends and outliers
• Easy comparison
• Supports decision making

Common Charts
• Line Chart
• Bar Chart
• Pie Chart
• Histogram
• Scatter Plot

Matplotlib is a popular Python library used for visualization.

Example:
import matplotlib.pyplot as plt
plt.plot([1,2,3],[10,20,30])
plt.show()
